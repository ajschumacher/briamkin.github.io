Python Is Awesome

I just want to say that compared to Javascript and PHP, Python is a beautiful language. I love the syntax that is used, especially the forced indentation. This really keeps code incredibly readable! I spent part of today looking at this quora post and was amazed at some of these amazing pieces of Python code that people have written! And this gives different 1 line Python scripts with very cool functionality. Though ironically, some of these aren't really done in the "Python way".

I'm sure other languages have similarly amazing scripts, but for today, I just want to extol the virtues of Python!
